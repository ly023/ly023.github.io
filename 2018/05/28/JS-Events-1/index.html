资源加载

发表于 2018-05-28 | 更新于 2018-06-06 | 分类于 JavaScript | 评论数： | 阅读次数：

load 事件

当一个资源及其依赖资源已完成加载时，将触发load事件。

支持该事件的HTML标签：

window.onload
document.onload

当整个html文档加载的时候触发，在body元素加载之前就开始执行。

image.onload

onload 事件在图片加载完成后立即执行

imgae.src = url应写在onload之后，避免图片缓存或加载太快导致图片加载完成但onload事件还未绑定的情况。

注意

从浏览器缓存加载网页，不会触发load事件。如果有每次加载（包括第一次加载）都要执行的代码，可以使用pageshow事件。另一方面，一旦使用unload事件，网页就不会被浏览器缓存。如果不希望这样，可以使用pagehide事件，保证每次网页卸载代码都会执行。

DOMContentLoaded 事件

DOM加载完后触发，IE9及以上，jQuery $(document).ready(function(){})核心就是DOMContentLoaded事件。

document.onreadystatechange 事件

当一个文档的readyState属性发生改变时，readystatechange事件会被触发。

可用于检测DOM是否加载完毕，当document.readyState === 'complete'时，表示DOM加载完成。但是如果页面中有iframe的话，会等到iframe中的所有资源加载完才会变成complete。

document.documentElement.doScroll 事件

IE特有，IE8以下，通过间隔调用document.documentElement.doScroll("left");可以检测DOM是否加载完成。当页面未加载完成时，该方法会报错，直到doScroll不再报错时，就代表DOM加载完成。
