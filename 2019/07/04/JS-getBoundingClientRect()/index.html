getBoundingClientRect()

发表于 2019-07-04 | 更新于 2019-07-06 | 分类于 JavaScript | 评论数： | 阅读次数：

getBoundingClientRect()方法返回元素的大小及其相对于视口的位置。该函数返回一个 DOMRect对象，包含了一组用于描述边框的只读属性——left、top、right和bottom，单位为像素。除了width和height 外的属性都是相对于视口的左上角位置而言的。

  • top：元素上边到视窗上边的距离

  • right：元素右边到视窗左边的距离

  • bottom：元素下边到视窗上边的距离

  • left：元素左边到视窗左边的距离

  • width：元素的宽度（等同于 right 减 left）

  • height：元素的高度（等同于 bottom 减 top）
